## 特徴

**300 以上、現在も増え続けており、小さな便利機能から大きな改善までゲーム全体を幅広くカバーするモジュール群**

## 導入方法

1. [プラグインリポジトリの配布ページ](https://github.com/AtmoOmen/DalamudPlugins) にアクセスし、リポジトリ URL を取得します
2. **Dalamud Settings - Experimental - Third-Party Plugins** を開き、リポジトリ URL を入力して、設定ページ右下の保存ボタンを押します
3. **Dalamud Plugins** を開き、`Daily Routines` を検索してインストールします
