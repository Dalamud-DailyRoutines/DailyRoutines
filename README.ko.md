## 특징

**300 개 이상, 지금도 계속 늘어나는 모듈로 작은 편의 기능부터 큰 개선까지 게임 전반을 폭넓게 지원합니다**

## 설치 방법

1. [플러그인 저장소 배포 페이지](https://github.com/AtmoOmen/DalamudPlugins) 에서 저장소 URL 을 확인합니다
2. **Dalamud Settings - Experimental - Third-Party Plugins** 를 열고 저장소 URL 을 입력한 뒤, 설정 페이지 오른쪽 아래의 저장 버튼을 누릅니다
3. **Dalamud Plugins** 를 열고 `Daily Routines` 를 검색하여 설치합니다
