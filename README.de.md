## Funktionen

**Mehr als 300 Module und weiterhin wachsend, die nahezu alle Bereiche des Spiels abdecken, von kleinen Komfortfunktionen bis hin zu größeren Verbesserungen**

## Installation

1. Öffnen Sie die [Veröffentlichungsseite des Plugin-Repositories](https://github.com/AtmoOmen/DalamudPlugins), um die Repository-URL zu erhalten
2. Öffnen Sie **Dalamud Settings - Experimental - Third-Party Plugins**, fügen Sie die Repository-URL ein und klicken Sie anschließend auf die Schaltfläche zum Speichern unten rechts auf der Einstellungsseite
3. Öffnen Sie **Dalamud Plugins**, suchen Sie nach `Daily Routines` und installieren Sie das Plugin
